# React-Koa2-Blog

## 基于React + Koa2的博客

前端部分:
1. 登录管理
2. 文章分类
3. 文章列表展示
4. 文章详情页展示
5. 权限管理
6. 文章标签管理
7. 发表、预览、保存文章

后端部分:
1. mongoose数据库操作
2. 路由管理
3. 身份验证
